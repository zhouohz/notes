# 实体

> HTML中的**预留字符**必须被替换为**字符实体**才能在页面中正确的显示

常用字符实体：

| 字符     | 描述                                     |
| -------- | ---------------------------------------- |
| `&nbsp;` | 一个空格（可以在页面中连续显示多个空格） |
| `&lt;`   | 小于符号                                 |
| `&gt;`   | 大于符号                                 |
| `&copy;` | 版权                                     |

需要用的时候查表，不需要记忆
